# HSC English Advanced — Shared Notes

Generic, shareable English Advanced study notes from an HSC student at Arthur Phillip High School (Year 12, 2026).

These are the **share-friendly** notes — generic study materials, quote banks, structural analysis, and reflection/essay frameworks. Personal practice writing and creative pieces are kept separate and not included in this repo.

## 🎯 How To Use These Notes

1. **Open in Obsidian** by copying the `shared/` folder into your Obsidian vault — wiki links will resolve automatically.
2. **Or browse on GitHub** — each `.md` file renders nicely with the table of contents up top.
3. **For each module**, start with the **Landing** file, then drill into the text-specific analyses, then finish with the **Collection Analysis** to nail the textual-integrity angle.
